# file-encryption

store keys safely inside your git repositories.

## Installing

```
npm i -g file-encryption
```

### USAGE

```
file-encryption help
```

```
file-encryption decrypt --path="./.file" --key="test-key"
file-encryption decrypt --path="./.file" --key="test-key"
```
